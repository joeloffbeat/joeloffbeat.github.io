# Art Gallery

Add subfolders here. Each subfolder becomes a tab in the Art Gallery overlay.

Example structure:
  _arts/
    digital-art/
      painting1.jpg
      painting2.png
    sketches/
      sketch1.jpg

Supported formats: jpg, jpeg, png, gif, webp

Folder names are converted to display names:
  digital-art  →  Digital Art
  my_sketches  →  My Sketches
